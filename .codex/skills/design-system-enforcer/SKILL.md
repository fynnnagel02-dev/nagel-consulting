# Design System Enforcer

When invoked:
1. Check for inconsistent:
   - button styles
   - spacing
   - typography
   - component usage
2. Replace inconsistent elements with existing patterns.
3. Remove unnecessary variations.
4. Ensure a small set of highly polished components.
5. Improve visual harmony and alignment.
6. Do not redesign everything — refine and unify.
