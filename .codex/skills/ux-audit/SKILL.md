# UX Audit Skill

When invoked:
1. Analyze layout structure from top to bottom.
2. Identify:
   - weak hierarchy
   - clutter
   - inconsistent spacing
   - unclear CTAs
3. Evaluate if the page feels premium or template-like.
4. Suggest concrete improvements only (no vague advice).
5. Always consider mobile layout and spacing.
6. Focus on high-impact changes, not micro-details.
